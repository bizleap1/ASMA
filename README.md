# Advait Share Market Academy

A modern, high-performance web application for India's biggest and most experienced stock market academy in Central India. The website is built with React and Vite, featuring a premium glassmorphism design, smooth interactive animations, and a fully responsive layout.

## ✨ Features

- **Premium UI/UX:** A polished, modern interface with glassmorphism effects, custom theme variables, and dynamic hover states.
- **Single Page Application (SPA):** Fast, seamless routing utilizing `react-router-dom` with a custom scroll-to-top handler.
- **Dedicated Pages:**
  - **Home:** Engaging hero section, market insights, and quick links.
  - **About Us:** The academy's journey, mission, vision, and core values.
  - **Services:** Comprehensive trading and financial services.
  - **Courses:** Detailed stock market training programs.
  - **Gallery:** Visual showcase of past events, seminars, and academy culture.
  - **Contact:** Information for getting in touch with experts.
  - **Legal:** Dedicated Privacy Policy and Terms & Conditions pages.
- **Mobile Responsive:** Fully optimized for all devices, including a smooth mobile navigation menu.

## 🛠 Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (with custom themes and animations)
- **Routing:** React Router v6
- **Icons & Graphics:** Embedded SVG icons, Unsplash placeholders, and curated local assets.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js installed on your machine.
- [Node.js](https://nodejs.org/en/) (v16 or higher is recommended)

### Installation

1. **Clone the repository** (or download the project folder):
   ```bash
   git clone <repository-url>
   cd ASMA
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open your browser and navigate to `http://localhost:5173` (or the port provided in your terminal).

## 📦 Building for Production

To create a production-ready build, run:
```bash
npm run build
```
This will generate optimized static assets in the `dist/` directory, which can be deployed to any static hosting provider (e.g., Vercel, Netlify, AWS S3, etc.).
